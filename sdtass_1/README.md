# Python Programming Tasks

This repository contains a collection of Python scripts demonstrating various programming concepts and solving basic algorithmic problems.

## Files and Descriptions

### 1. `good_sequence.py`
Calculates the minimum number of elements to remove from a sequence such that for every number `x` in the sequence, it appears exactly `x` times.

### 2. `max_split.py`
Given a balanced string of 'L' and 'R' characters, splits it into the maximum number of balanced substrings and prints them.

### 3. `minimize_number.py`
Determines the maximum number of times an array of numbers can be divided by 2 before an odd number is encountered.

### 4. `args and kwargs of python with proper examples.py`
A conceptual file that explains the usage of `*args` and `**kwargs` in Python functions with clear examples. 

### 5. `list and dictionary of python difference.py`
A conceptual document outlining the key differences between Python lists and dictionaries, covering structure, syntax, accessing elements, ordering, and mutability.

### 6. `take a number from the user and draw a pyramid using PyAutoGUI.py`
A script that takes an integer input from the user and automates the drawing of a staircase/pyramid pattern made of `#` characters in a separate text editor using the `pyautogui` library.

## Requirements
- Python 3.x
- `pyautogui` (Required for drawing the pyramid)

You can install `pyautogui` using:
```bash
pip install pyautogui
```

## How to Run
Most scripts can be executed directly from the terminal. For example:
```bash
python minimize_number.py
```
For `take a number from the user and draw a pyramid using PyAutoGUI.py`, ensure you have a text editor open and focused within 5 seconds after providing the input.
